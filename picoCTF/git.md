git 
===
```git init``` 
- 功能 : 初始化本地倉庫 

```git add [filename]``` 
- .代表所有資料夾東西
- 功能 : 把東西交到提交區

```git commit -m "版本資訊"``` 
- 功能 : 提交區的東西全部提交上去
- 備註要打""

```git commit -v```
- 功能 : 同上
- 打開代碼編輯器，顯示檔案修改的詳情及可以詳細地輸入備註內容

```git branch -a```
- 顯示所有版本

```git merge```
- 融合

```git reset ```
- 重設一下工作目錄的索引狀態

```git status -sb```
- 查詢當前工作目錄的詳細狀態

```git log```
- 歷史紀錄

```git checkout 資料```
- 只看那比資料
